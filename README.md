# Starforge Defender VII
Open this link: https://seabassmarin-beep.github.io/Starforge-Defender-VII/

**Starforge Defender VII** is an original browser-based space survival game by **seabassmarin-beep**.

Defend the Starforge, survive hostile waves, avoid meteors, use lock-on targeting, switch weapons, dash away from danger, and fight through advanced enemy AI, boss waves, gravity wells, and asteroid physics.

## Play

Open `index.html` in a browser.

If you are using a smartphone use it in landscape.

The game runs as a single HTML file and does not need any extra images, libraries, or downloads.

## Controls

| Action | Computer | Mobile / Touch |
|---|---|---|
| Move | Move mouse | Move finger |
| Shoot | Hold click, Space, or F | Hold tap |
| Lock target | Point at enemy/meteor, then press L | Point at enemy/meteor, then tap LOCK |
| Dash | Shift | ⚡ button |
| Change weapon | Q or E | WPN button |
| Pause | P, Esc, or Pause button | Pause button |
| Restart | R | Reload or tap/click after game over |
| Mute | M | Keyboard only |

## Features

- Mouse/finger follow movement
- Target lock-on system
- Multiple weapons: Pulse, Spread, Rail, and Plasma
- Enemy lasers
- Smarter enemy AI
- Boss waves
- Gravity wells
- Asteroid physics and limited fragmentation
- Radar/minimap
- Mobile buttons
- Sound effects
- Particle explosions, smoke, debris, screen flash, and cinematic space effects
